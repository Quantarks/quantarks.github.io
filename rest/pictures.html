REST: Usage and Dangers Picture Service

  • Quantarks™ API Docs
  • REST APIs
  • Picture Service

The Quantarks™ app displays photos of an element's usage. As of the moment, this is only available online. The images must be compressed, possibly by gzip to save the users' bandwidth.

Endpoints

The Pictures service can be accessed through POST requests sent to the endpoint 
https://quantarks.dev/api/pictures/

Parameters

  • token

    A string that must come with the request to authenticate an access to the API. This is not available to the public and can only be used within the Quantarks™ app. This token, as of the moment is generated inside the Quantarks™ app.

    Warning

    The API returns an ERR_CREDENTIAL_NONE error if no token has been provided.

    The API returns an ERR_CREDENTIAL_INVALID error if the token sent is invalid.

  • atomic_number

    An integer that represents the atomic number of the element of which the API will return photos of.

    Warning

    The API returns an ERR_PARAM_INVALID if the atomic number is either of the following:

    • A string that cannot be parsed into an integer
    • An integer that is less than 1 or MAX_ATOMIC_NUMBER (118 as of 2018)

    The API returns an ERR_PARAM_MISSING if the parameter has not been provided.

  • classification DEFAULT: "all"

    A string that indicates the type of images to be returned by the API, that is, if it is the dangers of the element, the uses, or both.

    Possible values: 

    [ "all", "danger", "use" ]

    Warning

    The API returns an ERR_PARAM_INVALID if the supplied argument is not in the list of possible values.

    Notice

    This parameter defaults to "all" if this parameter is not passed with the request.

Return Values

The API returns a JSON string that follows this format: 

[
    "errors": [],
    "value": [
        {
            "image": "https://quantarks.dev/assets/30312d687964726f67656e.jpg",
            "type": "dangers"
            "caption": "Hydrogen is a flammable gas, and can be found in the core of the stars reacting in nuclear fusion."
        },
        {
            "image": "https://quantarks.dev/assets/30312d687964726f67656e.jpg",
            "type": "dangers"
            "caption": "Hydrogen is a flammable gas, and can be found in the core of the stars reacting in nuclear fusion."
        }
    ]
]

If an error has occurred, the API returns a JSON string that follows this format: 

[
    "errors": [
        {
            "code": "ERR_PARAM_INVALID",
            "message": "The `atomic_number` provided (value: 119) was invalid."
        },
        {
            "code": "ERR_PARAM_INVALID",
            "message": "The `classification` provided (value: dangerous) was invalid."
        }
    ],
    "value": []
]
