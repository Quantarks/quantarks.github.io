Description

The QAudio is the wrapper class that represents audio content via the *.qmp3 file format.

Note

This class handles File I/O, and it unzips the *.qmp3 file. For handling ZIP files, take a look at the following libraries/classes:

  • GZipStream (a.k.a Unity.IO.Compression)
  • SharpZipLib

Constructors

  • private QAudio()

    Creates a new QAudio class.

    Note

    This is a private constructor, and as such cannot be accessed from outside the class. The only way to create a QAudio object is to make a call to QAudio.extract().
