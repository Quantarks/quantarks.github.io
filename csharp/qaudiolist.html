QAudioList is the class that manages audio assets as they are being used inside the app. Each UI element that can trigger audio playback must have a AudioKey string property that contains the lookup key for the audio content that it is supposed to play.For more information about how audio content is handled internally, check out How Audio Content Should Work in the App.

Properties

Items
public List<string> Items { get; private set; }

List of lookup keys for audio content.

Methods

GetQAudio(String)
public static QAudio GetQAudio(String key)
    throws FileNotFoundException, ArgumentOutOfRangeException;

Gets the QAudio object by looking it up using the provided key.

PARAMETERS:

  • string key

    The lookup key of the audio content to be played.

RETURNS: The new QAudio object that results from this operation.

THROWS:

  • FileNotFoundException

    Thrown when the app cannot find the audio or transcript file that has the same key as the one provided.

  • ArgumentOutOfRangeException

    Thrown when the key passed is not in the Items property of this object.
