Class Spec: OnlineRequest class.

  • Quantarks™ API Docs
  • Android Classes
  • OnlineRequest

Description

The OnlineRequest class handles all requests to the Online API, both synchronous and asynchronous.

Constructors

  • private OnlineRequest()

    Initializes a new OnlineRequest object.

Private Methods

This class has no private methods.

Note

This class can always be added as many private methods as necessary.

Public Methods

  • public get(RequestParameters rp)

    Sends a GET request to the online API with the request parameters.

    RETURNS: a RequestResult object that contains the response of the online API.

Static Methods

  • private findByNumber(int atomicNumber)

    Finds a chemical element by its atomic number

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findByName(String name)

    Finds a chemical element by its name

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findBySymbol(String symbol)

    Finds a chemical element by its atomic symbol

    PARAMETERS:

    symbol
    A String that represents the atomic symbol of the element to find. For example, the element Hydrogen has the symbol "H", and Argon has the atomic symbol "Ar"

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findByElectronConf(int[] electronConfiguration)

    Finds a chemical element by its electron configuration

    PARAMETERS:

    electronConf

    An array of integers representing the electron configuration of the element to find. For example, Argon has 3 electron shells, having 2 electrons on the first one, and 8 electrons each on the second and third one. They can be represented as an array of integers: [2, 8, 8].

    For more information on the electron configurations in this form, see this graphic (where the electron configurations are indicated in the top right corner of every element).

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.
