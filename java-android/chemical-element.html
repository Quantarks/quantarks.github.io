Class Spec: ChemicalElement class.

  • Quantarks™ API Docs
  • Android Classes
  • ChemicalElement

Description

The ChemicalElement class represents every chemical element in the periodic table, and provides utility functions for searching elements on demand.

Constructors

  • private ChemicalElement()
    private ChemicalElement(int atomicNumber)

    These constructors can only be accessed by members of the same class, that is, the findBy*() functions. See static methods for details.

    Warning

    Calling the constructor with no parameters must be followed by a call to ChemicalElement .setAtomicNumber() to initialize the object.

Private Methods

  • private setAtomicNumber()

    Sets this object's atomic number and reinitializes the object.

Public Methods

  • public getAtomicNumber()

    Gets the atomic number of this element.

    RETURNS: an int equal to the atomic number of this chemical element.

  • public getName()

    Gets the name of this element.

    RETURNS: a String object representing the element's name.

  • public getSymbol()

    Gets the atomic symbol of this element.

    RETURNS: a String object representing the element's symbol.

  • public getPlaces()

    Gets the places where this element can usually be found.

    RETURNS: a String object detailing this element's places of abundance.

  • public getHistory()

    Gets the history of this element.

    RETURNS: a String object detailing this element's history.

  • public getNameOrigin()

    Gets the origin of this element's name.

    RETURNS: a String object detailing the origin of this element's name.

  • public getUses()

    Gets the uses of this element.

    RETURNS: a List<String> object that contains this element's uses.

  • public getDangers()

    Gets the dangers of this element.

    RETURNS: a List<String> object that contains this element's dangers.

  • public getDiscoveryYear()

    Gets the uses of this element.

    RETURNS: a String object that contains this element's year of discovery.

    Info

    Instead of making it a Date, this method returns a String because the elements.json file contains things like "unknown", "18th century", and years.

  • public getDiscoverers()

    Gets the discoverers of this element.

    RETURNS: A List<String> containing the names of the discoverers of this element.

Static Methods

  • private findByNumber(int atomicNumber)

    Finds a chemical element by its atomic number

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findByName(String name)

    Finds a chemical element by its name

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findBySymbol(String symbol)

    Finds a chemical element by its atomic symbol

    PARAMETERS:

    symbol
    A String that represents the atomic symbol of the element to find. For example, the element Hydrogen has the symbol "H", and Argon has the atomic symbol "Ar"

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.

  • private findByElectronConf(int[] electronConfiguration)

    Finds a chemical element by its electron configuration

    PARAMETERS:

    electronConf

    An array of integers representing the electron configuration of the element to find. For example, Argon has 3 electron shells, having 2 electrons on the first one, and 8 electrons each on the second and third one. They can be represented as an array of integers: [2, 8, 8].

    For more information on the electron configurations in this form, see this graphic (where the electron configurations are indicated in the top right corner of every element).

    RETURNS: a ChemicalElement object representing the element. If nothing is found, returns null.
